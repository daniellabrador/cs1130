Conversion Whiz

Conversion Whiz is a conversion tool for temperature and length units. This tool is only limited to Fahrenheit, Celsius, meter, and foot units. This project is intended to showcase HTML and JavaScript basics and DOM manipulation. See source code.

This application requires a minimum of 600px screen width to be viewed properly (mobile phone on landscape works).
